HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags define aspects of HTML within '<' & '>'
  • Elements is everything contained within an open and closing tag
  • Child tags are contained within parent tags.
    These are typically organized by indents to make the code human readable.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • repo (repository): Folder that a project and its assests live in
  • org(organization): Owner of a project
  • commit: apply changes to a repot branch
  • branch: temporary copy of a repo you can edit without
    impacting the source code
  • fork: copy of a repot that is under ownership of a different org/individual
  • pull request: ask to add changes from fork/branch
  • merge: commit changes to main/master from fork/branch
